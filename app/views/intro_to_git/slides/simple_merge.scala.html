Simple Merge

  • The simplest way to merge is using the git merge branch command
  • The command creates a special commit
  • The difference to a normal commit is that it has two parents
    • The first parent is the current commit (HEAD)
    • The second parent is the commit request to merge
  • Important notes:
    • It works by joining commits, not branches
    • The branch of the merged commit is not erased
    • Which means that the branch can evolve in the future
    • And require more merges in the future
