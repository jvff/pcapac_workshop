Branch

  • In theory, a branch is a sequence of commits
  • (Branch) Third commit
  • Second commit
  • First commit
  • Commands:
    • git branch: list local branches
    • git branch -a : list all branches
    • git branch new_branch: create a new branch
    • git checkout new_branch: go to the new branch
    • git checkout branch2: create and go to a new branch
