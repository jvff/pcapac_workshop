Branch Example

  • git branch: list local branches (current branch has an asterisk next to it)
  • git status: also shows current branch
  • git branch -a : list all branches
  • git branch first_branch: create a new branch
  • git branch
  • git checkout branch2: create and go to a new branch
  • git branch
  • touch first_file: create a new file
  • git add first_file: add the new file to the index
  • git commit: add the new file to the index
  • git log --oneline --decorate --graph
  • git checkout -b second_branch: create and go to a branch
  • git branch
  • touch second_file
  • git add second_file
  • git commit
  • git log --oneline --decorate --graph
  • git checkout -b third_branch first_branch: create and go to a branch starting at a given commit
