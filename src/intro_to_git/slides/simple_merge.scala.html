Junção simples

  • O jeito mais fácil de fazer uma junção é através do comando git merge branch
  • O comando cria um commit especial
  • A diferença para um commit normal é que ele tem dois pais
    • O primeiro pai é o commit atual (HEAD)
    • O segundo pai é o commit juntado
  • Observações importantes:
    • Funciona juntando commits, não ramificações
    • A ramificação do commit não é apagada
    • O que significa que a ramificação pode evoluir no futuro
    • E precisar de mais junções no futuro
