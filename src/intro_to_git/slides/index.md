Index
=====

- A temporary area to store artifact copies
- The files in the index are used to create a new version
- Example:
    - `git status`
    - `git add ".gitignore"`
    - `git status`
    - `git diff --cached`
    - `git reset -- ".gitignore"`
    - `git status`
    - `git diff --cached`
    - `git diff`
