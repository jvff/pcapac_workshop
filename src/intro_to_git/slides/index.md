Índice
======

- Um espaço temporário para armazenar cópias de artefatos
- Os arquivos no índice são usados para criar uma nova versão
- Exemplo:
    - `git status`
    - `git add ".gitignore"`
    - `git status`
    - `git diff --cached`
    - `git reset -- ".gitignore"`
    - `git status`
    - `git diff --cached`
    - `git diff`
